열공메이트 개인정보처리방침

시행일: 2025년 2월 22일 | 최종 수정: 2026년 7월 23일

핵심 요약: 열공메이트는 기본적으로 개인정보를 수집하지 않으며, 모든 공부 기록은 사용자의 기기에만 저장됩니다. 선택 기능인 스터디룸을 켠 경우에 한해, 사용자가 정한 닉네임과 공부 상태가 같은 방 참여자에게 공유됩니다.

1. 수집하는 개인정보

열공메이트(이하 "앱")는 스터디룸 기능을 사용하지 않는 한 어떠한 개인정보도 수집하지 않습니다.

앱에서 생성되는 모든 데이터(공부 기록, 과목 정보, 설정, D-Day 등)는 사용자의 기기 내부 저장소(AsyncStorage)에만 저장되며, 외부 서버로 전송되지 않습니다. 공부 기록·통계·플래너 내용은 스터디룸을 사용하더라도 서버로 전송되지 않습니다.

2. 스터디룸 (선택 기능)

스터디룸은 방 코드를 아는 사용자들끼리 서로의 공부 상태를 실시간으로 보는 선택 기능입니다. 사용자가 직접 켜기 전에는 어떤 서버 통신도 하지 않습니다. 기능을 켜면 다음 정보가 서버에 저장되고 같은 방 참여자에게만 공유됩니다:

사용자가 정한 닉네임(실명 사용을 권장하지 않음) / 선택한 캐릭터 / 현재 공부 중 여부와 타이머 이름(과목명), 공부 모드, 시작·예정 종료 시각 / 오늘 공부시간 합계 / 익명 계정 식별자

이 정보는 Google Firebase(Google LLC, 서버 소재지 미국)에 저장됩니다. 이메일·전화번호 등 계정 가입 정보는 요구하지 않으며, 익명 인증만 사용합니다.

삭제: 방에서 나가면 공부 상태 정보가 삭제되고, 스터디룸 화면의 설정에서 "스터디룸 끄기"를 실행하면 서버에 저장된 내 정보(닉네임·공부 상태 등)가 즉시 모두 삭제됩니다. 앱을 삭제한 경우 잔존 데이터 삭제는 아래 문의처로 요청할 수 있습니다.

3. 수집하지 않는 정보

앱은 다음 정보를 수집하지 않습니다:

이름, 이메일, 전화번호 등 개인 식별 정보 / 위치 정보 / 사용 행태 분석 데이터 / 광고 식별자

4. 제3자 제공

스터디룸 정보의 저장·전송을 위해 클라우드 인프라 제공자인 Google LLC(Firebase)를 이용하는 것 외에, 어떠한 정보도 제3자에게 제공하지 않습니다.

5. 알림 권한

앱은 타이머 완료 알림을 위해 알림(Notification) 권한을 요청할 수 있습니다. 이 권한은 기기 내에서 로컬 알림을 표시하는 용도로만 사용되며, 푸시 알림 서버와 통신하지 않습니다.

6. 화면 밝기 접근

앱은 집중모드(화면 켜기 모드) 기능에서 화면 밝기를 일시적으로 조절합니다.

집중모드를 시작하면 화면이 어둡게 설정되어 눈의 피로를 줄여줍니다. 집중모드 종료 또는 앱 종료 시 화면 밝기는 원래 값으로 자동 복원됩니다. 이 기능은 기기 시스템 밝기(System Brightness)를 일시적으로 변경하는 것으로, 어떠한 데이터도 수집하거나 외부로 전송하지 않습니다.

7. 카메라 및 사진 접근 (오답노트 · 선택 기능)

오답노트에 문제 사진을 첨부하는 선택 기능을 사용할 때, 사용자가 직접 실행하는 경우에 한해 카메라와 사진 앨범에 접근합니다. 첨부한 사진은 사용자 기기 내부의 앱 전용 저장소에만 저장되며, 외부 서버로 전송되거나 제3자에게 제공되지 않습니다. 앱은 사진 앨범을 자동으로 스캔하지 않고, 사용자가 촬영하거나 선택한 사진만 저장합니다. 첨부한 사진은 해당 오답을 삭제하거나 설정의 "오답노트 사진 정리"에서 제거할 수 있으며, 앱을 삭제하면 함께 삭제됩니다.

8. 데이터 저장 및 삭제

공부 기록 등 모든 데이터는 기기의 로컬 저장소에 저장됩니다. 앱을 삭제하면 기기 내 모든 데이터가 함께 삭제됩니다. 앱 내 설정에서 데이터 초기화도 가능합니다. 스터디룸 서버 데이터 삭제는 2항을 참고하세요.

9. 아동 개인정보 보호

앱은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 스터디룸 닉네임에는 실명 등 개인 식별 정보를 사용하지 않기를 권장합니다.

10. 광고

앱에는 광고가 포함되어 있지 않습니다.

11. 변경 사항

본 개인정보처리방침이 변경될 경우, 앱 업데이트 또는 본 페이지를 통해 고지합니다.
