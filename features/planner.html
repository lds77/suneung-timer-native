계획과 실행이 저절로 연결되는 공부플래너

세운 계획과 실제 공부량이 자동으로 이어집니다. 달성률을 따로 계산할 필요가 없어요.

홈 › 플래너·과목

주간 시간표 그리드 iOS·Android

요일×시간 그리드에 과목·계획 블록을 배치하세요. 타이머로 공부한 시간이 계획 블록에 자동으로 채워지고, 달성률이 바로 보입니다.

  • 계획 대비 실제 공부량·달성률 자동 비교
  • 매주 반복 계획 vs 이번 주만 계획 구분
  • 계획 미루기(다른 날로 이동), 고정 일정 휴무 처리
  • 일간 · 월간 뷰 지원

고정 일과는 한 번만, 매주 자동 반복 iOS·Android

학교·식사·취침처럼 매주 똑같은 일과는 요일별로 한 번만 넣어두면 자동으로 반복됩니다. 처음이라 막막하다면 '기본으로 초기화'로 학교급에 맞는 시간표를 한 번에 채우고, 필요한 부분만 손보면 돼요.

  • 학교급 기본 시간표로 한 번에 채우기('기본으로 초기화')
  • '다른 요일에 복사'로 요일 설정을 빠르게
  • 고정 일정을 넣으면 '공부 가능 시간'이 자동 계산
  • 주간 뷰에서 옮긴 시간 배치는 그날에만 적용(반복 아님)

시험 D-Day 관리 iOS·Android

수능·기말고사·자격증 시험일을 등록하면 매일 자동으로 카운트합니다. 시험별로 준비 할 일 목록과 진행률을 함께 관리할 수 있어요.

  • 시험별 준비 할 일 + 진행률
  • 홈 화면 D-Day 위젯으로 잠금화면에서도 확인

과목별로 시간이 쌓여요 iOS·Android

과목마다 공부 시간이 자동으로 누적됩니다. 과목을 눌러 바로 타이머를 시작하고, 즐겨찾기와 주간 목표도 설정할 수 있어요.

  • 과목별 누적 시간 · 즐겨찾기
  • 자유/타이머로 과목 바로 시작 · 주간 목표
  • 홈 화면 '과목 바로 시작' 위젯으로 원터치 실행

검증된 학습법 프리셋 iOS·Android

어떻게 공부할지 고민될 땐 연구로 검증된 학습법을 프리셋으로 시작하세요. 학년에 맞춰 추천되고, 탭 한 번이면 여러 과목을 그대로 이어서 실행합니다.

  • 52-17 법칙 · 울트라디안 90 · 하드 스타트 등
  • 출처 표기 · 학년별 추천
